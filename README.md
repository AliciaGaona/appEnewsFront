# appEnewsFront

## Instrucciones para inicializar y correr proyecto

1. Instalar vue
   
```python
npm install vue
```

2. Instalar CLI de vue

```python
npm install -g @vue/cli
```

3. Instalar axios

```python
npm install axios
```

4.Levantar servidor vue

```python
npm run serve
```
